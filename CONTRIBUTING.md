# Contributing to Antigravity Status

Antigravity Status への貢献をご検討いただきありがとうございます。

## 開発の進め方

1. リポジトリをフォークします。
2. 新しいブランチを作成します (`git checkout -b feature/amazing-feature`)。
3. 変更をコミットします (`git commit -m 'Add some amazing feature'`)。
4. ブランチにプッシュします (`git push origin feature/amazing-feature`)。
5. プルリクエストを作成します。

## 規約

- コードの品質を維持するため、既存のスタイルに従ってください。
- 新しい機能を追加する場合は、README の更新も検討してください。
- セキュリティに関する重大な問題については、Issue を作成する前に直接報告してください。
